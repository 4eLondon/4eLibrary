# Input and Output — Go

## What is input and output

Go uses the `fmt` package for output and input. It is part of the standard
library — no install needed.

---

## Output

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
    fmt.Println("The answer is", 42)

    name := "London"
    fmt.Printf("Hello, %s!\n", name)
}
```

`Println` adds a newline automatically. `Printf` lets you format output
using verbs like `%s` for strings and `%d` for integers.

---

## Input

```go
package main

import "fmt"

func main() {
    var name string
    fmt.Print("Enter your name: ")
    fmt.Scan(&name)
    fmt.Println("Hello,", name)
}
```

Reading a number:
```go
var age int
fmt.Print("Enter your age: ")
fmt.Scan(&age)
fmt.Printf("Next year you will be %d.\n", age+1)
```

---

## Common mistakes

- `fmt.Scan` stops at whitespace — use `fmt.Scanln` or `bufio.Scanner` if you need a full line including spaces
- Always pass a pointer (`&variable`) to `fmt.Scan` — passing the value itself will not work

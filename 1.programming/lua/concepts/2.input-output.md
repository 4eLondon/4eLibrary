# Input and Output — Lua

## What is input and output

Lua uses `print` for basic output and `io.write` for more control.
Input is read with `io.read`.

---

## Output

```lua
print("Hello, World!")
print("The answer is", 42)

local name = "London"
print("Hello, " .. name .. "!")
```

`print` adds a newline and separates multiple values with a tab.
`io.write` gives you more control — no automatic newline:

```lua
io.write("Enter your name: ")
```

---

## Input

```lua
io.write("Enter your name: ")
local name = io.read()
print("Hello, " .. name .. "!")
```

Reading a number:
```lua
io.write("Enter your age: ")
local age = tonumber(io.read())
print("Next year you will be " .. (age + 1))
```

---

## Common mistakes

- `io.read()` returns a string — use `tonumber()` to convert to a number before doing maths
- `..` is the string concatenation operator in Lua — `+` does not join strings
- `print` adds a newline automatically — use `io.write` if you want output on the same line

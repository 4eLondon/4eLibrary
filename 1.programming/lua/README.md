## What is Lua

Lua is a lightweight, fast scripting language designed to be embedded inside
other applications. Created in Brazil in 1993, it was built to be small,
portable, and easy to integrate into larger programs. You will encounter Lua
anywhere a host application wants to offer scripting capability without
bundling a full language runtime. It is widely used in game development,
configuration, and developer tooling.

---

## Difficulty

**Beginner friendly.** Lua has a small number of concepts and clean syntax.
It is one of the easier languages to learn and is a good second language
after Python. The main adjustment is that Lua has some unusual conventions
— arrays start at index 1 instead of 0, and tables are the only data
structure, used for both arrays and dictionaries.

---

## What Lua is used for

| Field | Details |
|-------|---------|
| **Game development** | Roblox uses Lua as its primary scripting language. Many other game engines (LÖVE, CryEngine) use Lua for game logic. |
| **Editor configuration** | Neovim uses Lua for all plugin and configuration scripting, replacing the older VimScript. |
| **Embedded systems** | Lua's tiny footprint (under 300KB) makes it ideal for embedding in devices with limited resources. |
| **Game modding** | Many games expose a Lua API so players can write their own mods and scripts. |
| **Network devices** | Cisco, nginx (via OpenResty), and other networking tools use Lua for scripting rules and logic. |

---

## Key features

- **Tiny footprint** — the entire runtime is under 300KB
- **Fast** — one of the fastest scripting languages available
- **Tables as everything** — one data structure (the table) serves as arrays, dictionaries, objects, and modules
- **Easy to embed** — designed from the ground up to be integrated into C and C++ programs
- **Coroutines** — built-in cooperative multitasking

---

## Limitations

- **Niche use cases** — not a general-purpose language for building applications from scratch
- **Small ecosystem** — far fewer libraries than Python or JavaScript
- **1-indexed arrays** — arrays start at 1 not 0, which trips up developers from other languages
- **No standard OOP** — object-oriented patterns are possible but not built in

---

## What knowing Lua gets you

- Game development roles, especially in Roblox development (a large and growing platform)
- The ability to write Neovim plugins and configuration
- Embedded scripting skills useful in game modding and hardware work

## How to run Lua

**Terminal:**
```bash
lua filename.lua
```

**GUI (VS Code):**
Install the Lua extension. Use the integrated terminal or Code Runner.

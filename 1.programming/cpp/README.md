## What is C++

C++ is C with object-oriented programming, the standard template library,
and many additional features added on top. Created by Bjarne Stroustrup in
the early 1980s, it extends C while maintaining backwards compatibility.
It is one of the most powerful and complex languages in common use — and
one of the fastest. It powers game engines, graphics software, financial
trading systems, and anything else that demands maximum performance.

---

## Difficulty

**Very hard.** C++ inherits all of C's complexity and adds significant
additional complexity on top. The language has accumulated features over
decades and has many ways to do the same thing. Managing memory manually
while also understanding templates, operator overloading, move semantics,
and the rest of the language takes years to master. Not recommended as a
first language.

---

## What C++ is used for

| Field | Details |
|-------|---------|
| **Game engines** | Unreal Engine is written in C++. Most AAA game studios write their engine code in C++. |
| **Graphics and rendering** | Real-time 3D graphics, simulations, and visual effects pipelines. |
| **Financial systems** | High-frequency trading and financial modelling where nanoseconds matter. |
| **Browsers** — Chrome's rendering engine (Blink) and Firefox (Gecko) are written in C++. |
| **Compilers and tools** — LLVM, Clang, and many developer tools are built in C++. |
| **Embedded and systems** — anywhere C is used, C++ is also a viable option with more abstraction available. |

---

## Key features

- **Object-oriented** — classes, inheritance, polymorphism, and encapsulation
- **Templates** — generic programming that generates type-safe code at compile time
- **Standard Template Library** — vectors, maps, algorithms, and more built in
- **Zero-cost abstractions** — high-level features compile to the same efficient machine code
- **RAII** — resource management tied to object lifetimes, reducing memory leaks
- **Direct hardware access** — same low-level capabilities as C

---

## Limitations

- **Enormous complexity** — the language has accumulated decades of features and quirks
- **Unsafe by default** — manual memory management means bugs that are hard to find
- **Long compile times** — large C++ projects can take a very long time to compile
- **No package manager standard** — dependency management is fragmented compared to modern languages

---

## What knowing C++ gets you

- Roles in game development, graphics engineering, and high-performance computing
- A deep understanding of performance and systems programming
- The ability to work on some of the most complex and demanding software in existence

## How to run C++

**Terminal:**
```bash
g++ filename.cpp -o output  # compile
./output                    # run
```

**GUI (VS Code):**
Install the C/C++ extension. Use the play button or integrated terminal.

**GUI (CLion):**
JetBrains IDE for C++. Press `Shift+F10`. Handles CMake automatically.

# Input and Output — Bash

## What is input and output

Bash uses `echo` for output and `read` for input. These are among the first
commands you will use when writing shell scripts.

---

## Output

```bash
echo "Hello, World!"
echo "The answer is 42"

name="London"
echo "Hello, $name!"
echo "Hello, ${name}!"
```

Print without a newline:
```bash
echo -n "Enter your name: "
```

Print with escape characters (tabs, newlines):
```bash
echo -e "Line one\nLine two\tTabbed"
```

---

## Input

```bash
echo -n "Enter your name: "
read name
echo "Hello, $name!"
```

Read with a prompt on the same line:
```bash
read -p "Enter your name: " name
echo "Hello, $name!"
```

Read silently (for passwords):
```bash
read -s -p "Enter password: " password
echo ""
echo "Password stored."
```

---

## Common mistakes

- Always quote variables — `"$name"` not `$name` — unquoted variables break if they contain spaces
- `read` without a variable name stores input in `$REPLY` by default
- `echo` behaviour differs slightly between shells — `printf` is more portable for complex formatting

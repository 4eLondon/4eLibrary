# Input and Output — Java

## What is input and output

Java uses `System.out` for output and the `Scanner` class for input.
`Scanner` must be imported from `java.util`.

---

## Output

```java
System.out.println("Hello, World!");
System.out.println("The answer is " + 42);

String name = "London";
System.out.printf("Hello, %s!%n", name);
```

`println` adds a newline. `printf` formats output — `%s` for strings,
`%d` for integers, `%n` for a newline.

---

## Input

```java
import java.util.Scanner;

public class InputExample {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Enter your name: ");
        String name = scanner.nextLine();
        System.out.println("Hello, " + name + "!");

        scanner.close();
    }
}
```

Reading a number:
```java
System.out.print("Enter your age: ");
int age = scanner.nextInt();
System.out.println("Next year you will be " + (age + 1));
```

---

## Common mistakes

- Always close the `Scanner` when you are done — `scanner.close()`
- After calling `nextInt()` or `nextDouble()`, call `scanner.nextLine()` to consume the leftover newline before reading a string
- The `Scanner` class must be imported — `import java.util.Scanner;`

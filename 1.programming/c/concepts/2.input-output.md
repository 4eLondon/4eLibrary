# Input and Output — C

## What is input and output

C uses `printf` for output and `scanf` for input, both from the `stdio.h`
header. These are among the most fundamental I/O functions in programming.

---

## Output

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    printf("The answer is %d\n", 42);

    char name[] = "London";
    printf("Hello, %s!\n", name);

    return 0;
}
```

Common format specifiers: `%d` integer, `%f` float, `%s` string, `%c` character.

---

## Input

```c
#include <stdio.h>

int main() {
    char name[50];
    printf("Enter your name: ");
    scanf("%s", name);
    printf("Hello, %s!\n", name);

    return 0;
}
```

Reading a number:
```c
int age;
printf("Enter your age: ");
scanf("%d", &age);
printf("Next year you will be %d.\n", age + 1);
```

---

## Common mistakes

- Always pass a pointer to `scanf` — use `&variable` for numbers, arrays already decay to pointers
- `scanf("%s")` stops at whitespace — use `fgets` for reading full lines
- Never use `scanf` for strings without limiting the length — it can cause buffer overflows
- Always include `\n` in `printf` — C does not add newlines automatically

## What is C#

C# (pronounced C-sharp) is a modern, object-oriented language developed by
Microsoft as part of the .NET platform. First released in 2000, it was
designed to be clean, safe, and easy to use while being powerful enough for
enterprise software. It is the primary language for Unity game development
and .NET backend services, and it runs on Windows, macOS, and Linux through
the .NET runtime.

---

## Difficulty

**Intermediate.** C# is more structured than Python or JavaScript but less
dangerous than C or C++. The language is well-designed and the tooling —
particularly Visual Studio — is excellent. If you have learned Java, C# will
feel very familiar. The main adjustment is understanding the .NET ecosystem
and how projects are structured.

---

## What C# is used for

| Field | Details |
|-------|---------|
| **Game development** | Unity, the world's most widely used game engine, uses C# as its scripting language. Essential if you want to make games. |
| **Windows applications** | WPF and WinForms for desktop apps. If you are building software for Windows, C# is the natural choice. |
| **Web backend** | ASP.NET Core is a fast, modern web framework used at enterprise scale. Used by Stack Overflow and many large companies. |
| **Enterprise software** | Heavily used in corporate environments, finance, healthcare, and government systems running on Microsoft infrastructure. |
| **Cross-platform apps** | .NET MAUI lets you build mobile and desktop apps from a single C# codebase. |

---

## Key features

- **Modern syntax** — cleaner and more expressive than Java with features like LINQ, async/await, and pattern matching
- **Strong tooling** — Visual Studio is one of the best IDEs available
- **Garbage collected** — memory is managed automatically
- **Nullable reference types** — optional feature that eliminates null reference errors
- **LINQ** — a powerful query syntax for collections, databases, and XML built into the language
- **Unity integration** — deep integration with the Unity game engine

---

## Limitations

- **Microsoft ecosystem** — historically tied to Windows and Microsoft tools, though .NET Core has improved cross-platform support significantly
- **Slower than C/C++** — the runtime adds overhead compared to natively compiled languages
- **Learning curve with Unity** — C# itself is approachable, but Unity adds its own significant complexity

---

## What knowing C# gets you

- Unity game development — a large and growing industry
- Enterprise backend roles, especially in Microsoft-heavy organisations
- Windows desktop application development
- Access to the .NET ecosystem with mature tooling and libraries

## How to run C#

**Terminal:**
```bash
dotnet new console -n ProjectName
cd ProjectName
dotnet run
```

**GUI (Visual Studio):**
Press `F5` to build and run. The most fully featured IDE for C#.

**GUI (VS Code):**
Install C# Dev Kit. Use the play button or `dotnet run` in the terminal.

# Input and Output — JavaScript

## What is input and output

Output is how your program sends data somewhere visible.
Input is how your program receives data. Where this happens depends on
whether you are running JavaScript in a browser or in Node.js.

---

## Output

**Browser — console:**
```javascript
console.log("Hello, World!");
console.log("The answer is", 42);
```

**Browser — page:**
```javascript
document.body.innerHTML = "<p>Hello, World!</p>";
```

**Browser — popup:**
```javascript
alert("Hello, World!");
```

**Node.js — terminal:**
```javascript
console.log("Hello from Node.js");
process.stdout.write("No newline at end");
```

---

## Input

**Browser — popup:**
```javascript
const name = prompt("Enter your name:");
console.log("Hello, " + name);
```

**Browser — form element:**
```javascript
const input = document.getElementById("myInput");
const value = input.value;
console.log("You typed:", value);
```

**Node.js — readline:**
```javascript
const readline = require("readline");
const rl = readline.createInterface({ input: process.stdin });

rl.question("Enter your name: ", (name) => {
  console.log(`Hello, ${name}!`);
  rl.close();
});
```

---

## Common mistakes

- `prompt()` only works in the browser — it does not exist in Node.js
- All input from `prompt()` and form fields is a string — convert with `Number()` if you need a number
- `console.log()` is for development — remove or replace it before shipping to production

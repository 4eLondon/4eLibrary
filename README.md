# 01-programming

Welcome to the **4everCode** repository. This directory serves as a comprehensive collection of code examples, snippets, and implementations organized by programming language.

The goal of this library is to explore syntax, paradigms, and problem-solving approaches across a wide spectrum of languages—ranging from low-level system languages to high-level functional ones.

## 📂 Languages

This directory contains sub-folders for the following languages:

*   **Low-Level & Systems:**
    *   `assembly`
    *   `c`
    *   `cpp`
    *   `rust`
    *   `zig`
    *   `odin`

*   **Web & Scripting:**
    *   `python`
    *   `ruby`
    *   `lua`
    *   `javascript` (if present, or remove if not)
    *   `bash`
    *   `go`

*   **Enterprise & Object-Oriented:**
    *   `java`
    *   `csharp`

*   **Functional & Niche:**
    *   `haskell`
    *   `elixir`
    *   `ocaml`
    *   `nim`

## 📝 Usage

Each folder typically contains:
*   Source code files (`.rs`, `.go`, `.c`, `.py`, etc.)
*   Language-specific configuration or build files (e.g., `go.mod`, `Cargo.toml`)
*   Documentation specific to that implementation

## 🚀 Getting Started

To run examples from a specific language, navigate to the corresponding folder:

```bash
cd python
python hello_world.py
```

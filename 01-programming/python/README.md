## What is Python

Python is a general-purpose programming language designed to be readable and
simple. It reads almost like plain English which makes it one of the easiest
languages to learn from scratch. Created by Guido van Rossum and first
released in 1991, it has grown into one of the most widely used languages
in the world.

---

## Difficulty

**Beginner friendly.** Python is the most recommended first language for a
reason — the syntax is clean, the error messages are helpful, and you can
write something useful within hours of starting. There is very little
boilerplate before you can get things done.

---

## What Python is used for

| Field | Details |
|-------|---------|
| **Data science & ML** | The dominant language for data analysis, machine learning, and AI. Libraries like NumPy, Pandas, and PyTorch make it the industry standard. |
| **Web backend** | Django and FastAPI are two of the most popular backend frameworks. Instagram, Pinterest, and Spotify are all built on Python backends. |
| **Automation & scripting** | Automating repetitive tasks, file management, web scraping, and system administration. |
| **Cybersecurity** | Widely used for writing security tools, exploit scripts, and automation in penetration testing. |
| **Scientific computing** | Used heavily in academia and research for simulations, data processing, and visualisation. |
| **DevOps & tooling** | Configuration scripts, deployment automation, and infrastructure tools. |

---

## Key features

- **Readable syntax** — code looks clean and is easy to follow even for beginners
- **Huge standard library** — a massive collection of built-in modules covering almost everything
- **Third-party ecosystem** — over 400,000 packages on PyPI covering every use case imaginable
- **Interactive shell** — run and test code line by line in the Python REPL
- **Cross-platform** — runs on Windows, macOS, and Linux without modification
- **Garbage collected** — memory is managed automatically, no manual allocation needed

---

## Limitations

- **Speed** — Python is slower than compiled languages like C++ or Rust for CPU-heavy tasks
- **The GIL** — the Global Interpreter Lock limits true parallel thread execution
- **Mobile** — not a strong choice for native mobile app development
- **Not great for game development** — functional but not the right tool for performance-heavy games

---

## What knowing Python gets you

- Entry-level roles in data science, backend development, automation, and DevOps
- The ability to automate almost any repetitive task on your computer
- A foundation that makes learning other languages significantly easier
- Access to the largest and most active programming community in the world

## How to run Python

**Terminal:**
```bash
python3 filename.py
```

**GUI (VS Code):**
Install the Python extension. Open the file and click the play button (▷)
top right, or right-click and select `Run Python File`.

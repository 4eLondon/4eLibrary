# Input and Output — Python

## What is input and output

Input is how your program receives data from the outside world.
Output is how your program sends data back out.
In Python the two most common forms are reading from the keyboard and printing to the terminal.

---

## Output

The `print()` function sends text to the terminal.

```python
print("Hello, World!")
print("The answer is", 42)
```

You can format output using f-strings — the cleanest way to embed variables in text:

```python
name = "London"
age = 22
print(f"My name is {name} and I am {age} years old.")
```

---

## Input

The `input()` function pauses the program and waits for the user to type something and press Enter.
It always returns a string.

```python
name = input("Enter your name: ")
print(f"Hello, {name}!")
```

If you need a number, convert the string:

```python
age = int(input("Enter your age: "))
print(f"Next year you will be {age + 1}.")
```

---

## Common mistakes

- `input()` always returns a string — if you try to do maths with it without converting, you will get an error
- Forgetting to store the result of `input()` in a variable means you lose what the user typed

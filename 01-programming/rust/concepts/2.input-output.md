# Input and Output — Rust

## What is input and output

In Rust, output goes to stdout using the `println!` macro.
Input is read from stdin using the `std::io` module.
Rust is strict — reading input returns a `Result` type that you must handle.

---

## Output

```rust
fn main() {
    println!("Hello, World!");
    println!("The answer is {}", 42);

    let name = "London";
    println!("Hello, {}!", name);
}
```

Print without a newline:
```rust
use std::io::{self, Write};

print!("Enter your name: ");
io::stdout().flush().unwrap();
```

---

## Input

```rust
use std::io;

fn main() {
    let mut input = String::new();

    println!("Enter your name: ");
    io::stdin().read_line(&mut input).expect("Failed to read line");

    let name = input.trim();
    println!("Hello, {}!", name);
}
```

Reading a number:
```rust
use std::io;

fn main() {
    let mut input = String::new();
    io::stdin().read_line(&mut input).expect("Failed to read");

    let num: i32 = input.trim().parse().expect("Please enter a valid number");
    println!("Double that is {}", num * 2);
}
```

---

## Common mistakes

- `read_line` appends to the string — always use a fresh `String::new()` for each read
- Input includes a trailing newline — always call `.trim()` before using the value
- `.parse()` returns a `Result` — handle it with `.expect()` or proper error handling

# Input and Output тАФ C#

## What is input and output

C# uses `Console.WriteLine` for output and `Console.ReadLine` for input.
Both are part of the `System` namespace which is included by default.

---

## Output

```csharp
Console.WriteLine("Hello, World!");
Console.WriteLine("The answer is " + 42);

string name = "London";
Console.WriteLine($"Hello, {name}!");
Console.WriteLine($"Pi is approximately {3.14159:F2}");
```

`WriteLine` adds a newline. `Write` does not. The `$""` syntax is called
an interpolated string and is the cleanest way to embed variables.

---

## Input

```csharp
Console.Write("Enter your name: ");
string name = Console.ReadLine();
Console.WriteLine($"Hello, {name}!");
```

Reading a number:
```csharp
Console.Write("Enter your age: ");
int age = int.Parse(Console.ReadLine());
Console.WriteLine($"Next year you will be {age + 1}.");
```

Safer number reading:
```csharp
Console.Write("Enter your age: ");
if (int.TryParse(Console.ReadLine(), out int age)) {
    Console.WriteLine($"Next year you will be {age + 1}.");
} else {
    Console.WriteLine("That was not a valid number.");
}
```

---

## Common mistakes

- `Console.ReadLine()` always returns a string тАФ convert with `int.Parse()` or `int.TryParse()` for numbers
- `int.Parse()` throws an exception if the input is not a number тАФ use `TryParse` for safer code
- `Console.ReadLine()` can return `null` тАФ check for null if the program might receive no input

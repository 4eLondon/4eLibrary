## What is Java

Java is a compiled, object-oriented language that runs on the Java Virtual
Machine (JVM). Created by Sun Microsystems in 1995 under the slogan "write
once, run anywhere", Java code compiles to bytecode that runs on any device
with a JVM installed. It is one of the most widely deployed languages in
enterprise software and was the primary language for Android development for
many years.

---

## Difficulty

**Intermediate.** Java is more verbose than Python or JavaScript — you will
write more code to accomplish the same thing. It also requires understanding
object-oriented programming from the start since almost everything in Java
is a class. The concepts are well-documented and the ecosystem is mature,
making it very learnable with patience.

## What Java is used for

| Field | Details |
|-------|---------|
| **Enterprise software** | Banks, insurance companies, and large corporations run enormous amounts of Java. Spring Boot is the dominant backend framework. |
| **Android development** | Java was the original Android language. Kotlin has largely replaced it but Java is still supported and widely used. |
| **Big data** | Hadoop and Apache Spark — two of the most widely used big data tools — are written in Java. |
| **Web backends** | Spring, Jakarta EE, and other frameworks power high-traffic Java web applications. |
| **Desktop applications** | Cross-platform desktop apps using JavaFX or Swing. |

---

## Key features

- **Platform independent** — runs on any OS that has a JVM
- **Strongly typed** — all types are declared explicitly, catching many errors at compile time
- **Mature ecosystem** — decades of libraries, frameworks, and tooling
- **Garbage collected** — memory is managed automatically
- **Multithreading** — built-in support for writing concurrent applications
- **Backwards compatible** — old Java code still runs on modern JVMs

---

## Limitations

- **Verbose** — requires significantly more code than modern languages for the same result
- **Slow startup** — JVM startup time is noticeable compared to compiled native languages
- **Boilerplate heavy** — getters, setters, and constructors add a lot of repetitive code
- **Less popular for new projects** — Kotlin has largely replaced Java for Android, and Go/Python are preferred for new backend projects

## How to run Java

**Terminal:**
```bash
javac Filename.java   # compile — note the capital first letter
java Filename         # run — no .java extension
```

**GUI (IntelliJ IDEA):**
Click the green play button next to `main` or press `Shift+F10`.

**GUI (VS Code):**
Install the Extension Pack for Java. Play button appears above `main`.

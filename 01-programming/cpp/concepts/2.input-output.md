# Input and Output — C++

## What is input and output

C++ uses `std::cout` for output and `std::cin` for input, both from the
`<iostream>` header. These are cleaner than C's `printf`/`scanf` and
work with the type system automatically.

---

## Output

```cpp
#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    std::cout << "The answer is " << 42 << std::endl;

    std::string name = "London";
    std::cout << "Hello, " << name << "!" << std::endl;

    return 0;
}
```

`<<` is the stream insertion operator — it sends data into the output stream.
`std::endl` flushes the buffer and adds a newline. `"\n"` just adds a newline
and is slightly faster.

---

## Input

```cpp
#include <iostream>
#include <string>

int main() {
    std::string name;
    std::cout << "Enter your name: ";
    std::cin >> name;
    std::cout << "Hello, " << name << "!" << std::endl;

    return 0;
}
```

Reading a full line with spaces:
```cpp
std::string fullName;
std::cout << "Enter your full name: ";
std::getline(std::cin, fullName);
std::cout << "Hello, " << fullName << "!" << std::endl;
```

---

## Common mistakes

- `std::cin >>` stops at whitespace — use `std::getline` for full lines
- After using `std::cin >>` to read a number, call `std::cin.ignore()` before `std::getline` to consume the leftover newline
- Using `using namespace std;` removes the need for `std::` — common in examples but avoid in large projects
